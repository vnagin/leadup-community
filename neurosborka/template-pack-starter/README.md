# Стартовый pack шаблонов AI-фрилансера

> Пять рабочих шаблонов для запуска AI-практики: договор, NDA, ТЗ, питч-дек, прайс-калькулятор. Каждый — простой Markdown / DOCX / Google Sheet, без юридических хитростей и без интеграций. Цель — снизить порог входа для участника, который вчера сказал «не знаю с чего начать».

## Состав

| # | Файл | Назначение | Источник | Готовность |
|---|---|---|---|---|
| 1 | `01-contract-ai-services-ru.md` | Договор оказания AI-услуг (физлицо / самозанятый / ИП) — 1 страница | Open-source шаблон ФЛ-ЮЛ + ревизия Кати | draft |
| 2 | `02-nda-mutual-ru.md` | Взаимное NDA (стандартное, под фриланс-проект) | Open-source + ревизия | draft |
| 3 | `03-tz-template-ai-automation-ru.md` | ТЗ на AI-автоматизацию (n8n / make / GPT-инфраструктура) | Собран из 3-4 проектов Димы (AutomationEngineer) | draft |
| 4 | `04-pitch-deck-ai-freelancer-ru.md` | Питч-дек для первой встречи с клиентом (10 слайдов, текст-only) | Шаблон LeadUp + аудит топ-кейсов | draft |
| 5 | `05-price-calculator-ai-services-ru.md` | Прайс-калькулятор AI-услуг (часы × ставка + риск-надбавка + типовые пакеты) | Excel-расчёт от Димы | draft |

## Механика «внеси свой → упоминание в дайджесте»

1. Любой участник Free может прислать **свой шаблон / свою улучшенную версию** в личку Кате (@leadupai).
2. Катя проверяет (формат, отсутствие токсичной/нерабочей механики), добавляет в репозиторий GitHub `vnagin/leadup-community` с пометкой автора (PR / commit).
3. Включает в **ближайший понедельник-дайджест** строкой «новый шаблон от {имя}: {название} → ссылка». 1–2 строки, без пафоса.
4. Если шаблон применил кто-то ещё и у него получилось — это становится поводом для «кейса участника недели» (см. фаза 3 §3.1).

## Что нужно от CEO (одно ревью пакетом)

- ✅ на структуру пакета (5 файлов, охват достаточный для первого клиента).
- ✅ на 5 драфтов (1 страница договора / 1 страница NDA / 2-3 страницы ТЗ / 10 слайдов / Excel-калькулятор).
- ✅ на лицензию (CC BY 4.0 с упоминанием Нейросборки) и на хостинг (**GitHub `vnagin/leadup-community`**, прецедент n8n-pack [LEA-1408](/LEA/issues/LEA-1408)).
- ✅ на механику «внеси свой → упоминание в дайджесте».
- Опционально: правки/комментарии прямо в файлах — Катя интегрирует commit-ом.

**Бюджет CEO time:** 30 мин разово (ревью 5 драфтов + механика).

## Что не делаем в первой версии

- Скачивание — для участника одной ссылкой (raw / релиз-zip), без требования знать git. GitHub только как хранилище и витрина авторства, не как порог входа.
- НЕ переводим на сложные конструкторы (Tilda-сборки, Notion-публикации).
- НЕ вкладываем туда платные T1-материалы. Это подарок за факт пребывания в Free, а не воронка.
- НЕ добавляем партнёрские реферальные ссылки в шаблоны.
