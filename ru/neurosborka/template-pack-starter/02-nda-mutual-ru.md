# Взаимное NDA для AI-проектов

## Шаблон

```
СОГЛАШЕНИЕ О КОНФИДЕНЦИАЛЬНОСТИ № __________
(Mutual Non-Disclosure Agreement)

г. ____________                                          «___» ___________ 20___ г.

Сторона 1: ____________________________________________________________________
Сторона 2: ____________________________________________________________________

(вместе — «Стороны»)

1. ПРЕДМЕТ
1.1. Стороны обмениваются конфиденциальной информацией в связи с обсуждением и/или исполнением проекта по разработке AI-решения. Цель этого Соглашения — взаимная защита такой информации.

2. КОНФИДЕНЦИАЛЬНАЯ ИНФОРМАЦИЯ
2.1. Под «Конфиденциальной информацией» понимается любая нематериальная или материальная информация, переданная одной Стороной другой, включая (но не ограничиваясь):
- бизнес-процессы, стратегии, планы;
- коммерческие условия, ценообразование;
- технические сведения о системах, инфраструктуре, API-ключах;
- данные клиентов / контрагентов передающей Стороны;
- черновики, прототипы, исходный код, промпты, датасеты.
2.2. **Не является** конфиденциальной информация, которая:
- была публично известна на момент передачи;
- получена Стороной из других законных источников;
- была разработана получающей Стороной независимо;
- раскрытие которой требуется по закону.

3. ОБЯЗАТЕЛЬСТВА СТОРОН
3.1. Каждая Сторона обязуется:
- использовать Конфиденциальную информацию **исключительно** для цели Проекта;
- не передавать её третьим лицам без письменного согласия другой Стороны;
- применять разумные меры для её защиты (не менее, чем для своей собственной информации аналогичной чувствительности);
- ограничить доступ к ней только теми сотрудниками/подрядчиками, которым она нужна, и предупредить их об обязательствах конфиденциальности.

4. AI-ИНСТРУМЕНТЫ И ОБРАБОТКА ИНФОРМАЦИИ
4.1. Если для исполнения Проекта Сторона использует **AI-инструменты сторонних провайдеров** (OpenAI, Anthropic, Google Gemini, локальные модели и т.п.) — она обязуется:
- использовать **API/корпоративные планы**, в которых данные не используются для обучения моделей по умолчанию (для OpenAI — API-уровень; для Anthropic — стандартный API);
- НЕ передавать в публичные веб-чаты (ChatGPT.com, claude.ai, Gemini.google.com без enterprise-плана) Конфиденциальную информацию другой Стороны без её отдельного письменного согласия;
- удалять промежуточные артефакты (логи запросов, кэши) по запросу другой Стороны.

5. СРОК
5.1. Настоящее Соглашение действует ____ лет с даты подписания. Обязательства по сохранению конфиденциальности сохраняются ещё ____ лет после прекращения Соглашения.

6. ОТВЕТСТВЕННОСТЬ
6.1. За нарушение обязательств по настоящему Соглашению нарушившая Сторона возмещает другой Стороне доказанные убытки, причинённые таким нарушением.
6.2. Стороны соглашаются, что не каждое нарушение можно компенсировать денежно, и сохраняют право требовать обеспечительных мер (запрета дальнейшего разглашения).

7. ПРОЧЕЕ
7.1. Все споры — путём переговоров; при недостижении согласия — в суде по месту нахождения Истца.
7.2. Изменения — только в письменной форме.
7.3. Соглашение составлено в 2 экземплярах.

ПОДПИСИ:
Сторона 1: ________________ / ___________ /
Сторона 2: ________________ / ___________ /
```

## Чек-лист

- [ ] Заполнить реквизиты Сторон.
- [ ] Срок действия (по умолчанию: 3 года + 5 лет post-term для критичных проектов; для лёгких — 1+2 года).
- [ ] Если в Проекте используются AI-инструменты — пункт 4 обязателен.
- [ ] Согласовать с другой Стороной (типичные правки — срок и формулировка п. 6).

## Что НЕ покрывает

- Передача персональных данных физлиц (нужно дополнительно Поручение по 152-ФЗ).
- Использование облачных AI-провайдеров с серверами вне РФ — отдельный пункт о трансграничной передаче данных, если применимо.
- Юрисдикция вне РФ — нужен local law expert.
