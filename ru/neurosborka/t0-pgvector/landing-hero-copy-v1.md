# Hero copy лендинга T0 PGVector

## H1 (заголовок)
**PGVector + локальный AI: гайд для AI-фрилансера**

## Подзаголовок
Сделайте первый рабочий RAG-проект на бесплатной базе — без Pinecone, без облака за рубежом и без того, чтобы быть программистом. Гайд + 4 готовых n8n-workflow.

## Буллеты ценности (3 шт, под hero-визуалом)
- **Бесплатно и у клиента.** PGVector — расширение обычного Postgres. Данные остаются в вашем (или клиентском) контуре — аргумент по деньгам и по 152-ФЗ.
- **4 готовых workflow.** Загрузка документов, поиск по смыслу, Telegram-бот «спроси у документов», автотегирование. Импортировали — подключили ключи — работает.
- **За один вечер.** По шагам, без кода. В конце — чек-лист «я готов брать заказ на RAG».

# Сегментный месседж (40+ исполнители, не программисты)

**Боль:** «AI-автоматизация — это, наверное, для программистов. Я не потяну техническую часть, а заказы на ботов по документам уже спрашивают.»

**Месседж лендинга снимает именно это:**
- Слово «программист» убрано из требований — везде «импортировали → подключили ключи → работает».
- Никакого облачного vendor-страха — «данные у вас / у клиента».
- Понятная награда за вечер: «первый работающий RAG» + чек-лист готовности брать заказ.
- Тон — спокойный, без хайпа и без «успей-купи». Ценность вперёд, продажа PRO/Нейромастерской — мягким P.S. в конце гайда, не на лендинге.

**Чего НЕ делаем на лендинге:**
- Не обещаем «заработок Х₽ за неделю».
- Не ставим публичных дедлайнов / счётчиков.
- Не просим телефон — только email (152-ФЗ red line: телефоны = RED, см. ADR-003).
